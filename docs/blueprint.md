# **App Name**: BookNova

## Core Features:

- Homepage Hero Section: Full-width hero banner with a futuristic gradient background, title, subtitle, and a search bar or CTA button.
- Popular Audiobooks Carousel: Horizontal scrollable carousel showcasing popular audiobooks with rounded cards, cover images, titles, authors, prices, and play icon overlays.
- Latest E-Books Grid: A 3 or 4-column grid displaying the latest e-books with clean cards including book cover, title, author, price, and a view button.
- Categories Grid: A grid of rounded icons for different book categories such as Self-Help, Literature, etc., with minimalistic and soft glowing icons.
- Dark/Light Theme Toggle: A theme switch at the top-right to toggle between full dark mode and light mode.
- Responsive Layout: A mobile-first layout with a fully responsive grid, ensuring optimal viewing experience across different devices.
- Blank Page Routes: Creation of blank placeholder pages for navigation such as /books, /audiobooks, /categories, /profile, /login, /register.

## Style Guidelines:

- Primary color: Neon blue (#4287f5) for a futuristic and vibrant look.
- Background color: Dark gray (#222222) desaturated from primary color, fitting the dark scheme.
- Accent color: Purple (#8742f5), analogous to the primary but with contrasting brightness and saturation, for highlights and interactive elements.
- Body and headline font: 'Vazirmatn', a modern Persian-friendly sans-serif, or 'Inter' (sans-serif) if Vazirmatn is unavailable.
- Use minimalistic icons with a soft glowing effect for categories and other UI elements.
- Implement a mobile-first layout with a responsive grid, ensuring adaptability across various screen sizes. Employ bold section spacing to create a clean and well-organized visual hierarchy.
- Apply smooth transitions and hover effects with light glass blur on header and cards, enhancing the futuristic and interactive feel.
